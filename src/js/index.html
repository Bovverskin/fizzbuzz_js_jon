Fizz Buzz Description

Fizz buzz is a group word game for children to teach them about division. Players take turns to count incrementally, replacing any number divisible by three with the word "fizz", any number divisible by five with the word "buzz", and any number divisible by fifteen with the word "fizzbuzz". Here is a simple calculator which will tell you whether that number is a "fizz", a "buzz" or a "fizzbuzz". Simply input your number and hit the "Push Me" button.
